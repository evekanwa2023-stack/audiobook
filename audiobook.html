ตั้งค่าครั้งเดียว ใช้ได้ทุกอุปกรณ์ (PC · iPhone · Android)

1

สร้าง GitHub account & repo

ไปที่ github.com → Sign up ฟรี → New repository → ตั้งชื่อ audiobook → Public → Create

2

อัปโหลด audiobook.html

ในหน้า repo → คลิก Add file → Upload files → ลาก audiobook.html ใส่ → Commit changes

3

เปิด GitHub Pages

Settings → Pages → Source: Deploy from branch → main → / (root) → Save
รอ 1-2 นาที URL จะเป็น: https://USERNAME.github.io/audiobook/audiobook.html

4

ตั้งค่า Google Cloud + ใส่ URL

ไปที่ console.cloud.google.com → สร้างโปรเจค → เปิด Drive API → สร้าง OAuth 2.0 Client ID (Web app) → Authorized JS origins ใส่: https://USERNAME.github.io

📱 iPhone: เปิด Safari → พิมพ์ URL GitHub Pages → บุ๊คมาร์คไว้ที่หน้าจอหลักได้เลย (Add to Home Screen)
1

ตั้งค่า Google Cloud

ไปที่ console.cloud.google.com → สร้างโปรเจค → เปิด Drive API → สร้าง OAuth Client ID (Web app) → Authorized JS origins: http://localhost:8080

2

เปิดโดยดับเบิ้ลคลิก start_audiobook.bat

ต้องมี Python ติดตั้งไว้ในเครื่อง (python.org) — บน PC เท่านั้น iPhone เข้าไม่ได้
